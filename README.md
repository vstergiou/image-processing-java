# Image Proccecing with Java

Image Processing created purely in Java converting every image format to [PPM][ppmlink] and [YUV][yuvlink] image formats.



## Features

* Grayscale
* Increase size
* Decrease size
* Rotate Clockwise
* Equalize Histogram
* Stacking Algorithm
